# ShipmentDemo

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 17.0.10.

## Steps to run the application on server
 1. download the zip code or clone the repo using link `https://github.com/Akshaykumar4524/shipment-demo.git`.
 2. navigate to project directory and install the necessary packages by `npm install`
 3. after installing run `npm start` to run the application
 4. after running you need to open `http://localhost:4200/` to see the running app
